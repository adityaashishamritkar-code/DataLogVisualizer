This is a custom logic programming environment. It is based on the syntax and functionality of Datalog.

Key features:
1. Recursive Inference Engine: It is able to resolve complex rules and identify infinite recursion
2. Visualization: After entering a set of rules, if you enter "visualize" into the terminal, you will be redirected to a url with a representation of the dependency graph
3. Negation as failure: Allows the compiler to reason about "if something can't be proven as true, then it is false"

Tools:
KrokiAPI - for the visualization
